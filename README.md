# 学生信息管理系统

一个基于 C++ 面向对象设计的命令行学生信息管理系统，支持学生基本信息的增删改查及文件持久化存储。

## 功能特性

- **数据初始化**：程序启动时自动从 `students.csv` 加载数据到内存
- **添加记录**：录入新学生信息，自动校验学号唯一性、性别合法性、日期合法性
- **修改记录**：根据姓名定位并修改学生信息，支持回车保留原值
- **显示记录**：表格形式输出所有学生信息，年龄自动计算
- **查找记录**：支持按学号精确查找和按姓名查找
- **删除记录**：支持按学号或姓名删除，删除前需确认
- **保存文件**：将数据以 CSV 格式持久化到磁盘
- **完善的输入校验**：学号唯一性、日期合法性（含闰年判断）、性别限制等

## 技术栈

- **语言**：C++17
- **开发环境**：Visual Studio 2022 (MSBuild v143)
- **容器**：`std::vector<Student>`
- **文件格式**：CSV（逗号分隔值）
- **字符编码**：UTF-8

## 编译与运行

### 方法一：Visual Studio

1. 打开 `ContactA.sln`
2. 选择目标配置（Debug/Release，x64/Win32）
3. 按 `Ctrl+Shift+B` 编译，或按 `F5` 运行

### 方法二：命令行（MSBuild）

```bash
# 编译
msbuild ContactA.sln /p:Configuration=Debug /p:Platform=x64

# 运行
./x64/Debug/ContactA.exe
```

### 编码说明

本项目使用 UTF-8 编码，项目文件已配置 `/utf-8` 编译选项。若在其他环境编译出现中文乱码，请确保源文件以 UTF-8 编码保存。

## 数据文件格式

`students.csv` 采用逗号分隔格式，每行一条记录：

```csv
姓名,学号,性别,年,月,日,宿舍号
张三,2021001,男,2002,5,15,3-101
李四,2021002,女,2003,8,22,3-102
```

## 类设计

| 类名 | 职责 |
|------|------|
| `Date` | 日期内嵌子对象，提供日期校验、闰年判断、年龄计算、运算符重载 |
| `Student` | 数据实体，包含基本信息和内嵌 `Date` 对象，重载 `>>`/`<<` 运算符 |
| `StudentManager` | 业务管理类，封装 `vector` 容器和文件 I/O，实现增删改查 |
